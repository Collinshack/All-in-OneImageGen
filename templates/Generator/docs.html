API Documentation

GET /api/v1/images/

Retrieves a list of generated images and their corresponding prompts.

Request

HTTP Method Endpoint Parameters
GET /api/v1/images/ N/A

Response

The API response will contain a JSON array of image and prompt objects.

Field Type Description
prompt string The prompt that generated the image.
generated_image string The URL of the generated image.

Example Response: 

[
  {
      "prompt": "an african goat",
      "generated_image": "/media/images/an_african_goat-image.jpg"
  },
  {
      "prompt": "a kitten in spain",
      "generated_image": "/media/images/a_kitten_in_spain-image.jpg"
  }
]

Response Codes

HTTP Status Code Description
200 OK The request was successful, and the response contains the list of images and prompts.

Notes

  • The generated_image field in the response contains the relative URL of the image. You can append it to your base URL or media URL to retrieve the actual image.
  • Make sure to handle the media URL correctly to display the images in your application.

Python Example

  
    import requests

    # Send GET request to the API endpoint
    url = 'http://127.0.0.1:8000/api/v1/images/'
    response = requests.get(url)

    # Check if the request was successful
    if response.status_code == 200:
        # Extract the JSON data from the response
        data = response.json()

        # Process the data
        for item in data:
            prompt = item['prompt']
            generated_image_url = item['generated_image']
            # Do something with the prompt and generated image URL

    else:
        # Handle the error case
        print(f"Request failed with status code {response.status_code}")
